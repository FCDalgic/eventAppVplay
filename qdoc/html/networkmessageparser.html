Detailed Description

The NetworkMessageParser class parses the received network reply from server.

The PreviewWindow class inherits QObject. The class receives raw server response and then tries to parse it. If input can be parsed succesfully, related signal will be emited according to desired data type. ...

[slot] void NetworkMessageParser::parseNetworkMessage(QString pMessage)

Tries to parse given input message. pMessage is the input message observed from QNetworkReply

[signal] void NetworkMessageParser::placesMessageParsed(QString pType, QList<PlaceContainer *> pPlaces)

This signal is emited when input message is succesfully parsed and its type is places_view. pPlaces; is the obtained place list which are ready to visualize in QML. pType; is the type of event observed from input.

[signal] void NetworkMessageParser::requestMessageParsed(QString pType, QList<EventContainer *> pEvents)

This signal is emited when input message is succesfully parsed and its type is events pEvents; is the obtained event list which are ready to visualize in QML. pType; is the type of event observed from input.
