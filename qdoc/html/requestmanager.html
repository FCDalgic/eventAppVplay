[slot] void RequestManager::onEventMessageParsed(QString pType, QList<EventContainer *> pEvents)

Triggers when we get event request response from server. Then emits the appropriate signal which is related with transmitted event type to server. Each emitable signal are corresponding to related container which are used in QML elements pEvents; List of events received from server reply. pType; Received request type
